What's a JSON

JSON aren't so bad!

A JSON is a standardized method of representing data and it looks something like this

{  "results": {
        "firstKey": "This is a string",
        "secondKey": "This is a number",
        "thirdKey": ["element0", "element1", "element3"],
        "question": "Do you get it?"
    }
}

JSON are built in a "key": "value" format
